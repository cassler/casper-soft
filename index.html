Casper Soft

A fork of the Casper theme for Ghost used on Darin.co

View on GitHub Download .zip Download .tar.gz

Casper Soft is a slight modification of the basic Casper theme provided with Ghost 0.7.1.

Special Layouts for Link Lists

Any post tagged "quickbits" will be treated differently on the index. The post title is presented subtlely and the full content of the post is displayed rather than an excerpt. It is recommended that you keep the content of these posts exclusively to unordered lists. A featured image is not displayed on the index but is available on individual archive pages.

Bigfoot.JS Footnotes

The default footnote format is augmented and styled using BigFootJs. Rather than a superscript number you'll see a nice inline symbol which displays your footnote/sidenote inlined. If javascript is disabled, everything works as usual.
